## 系统简介

### 什么是 OpenClaw-Admin？

OpenClaw-Admin 是一个基于 Vue 3 构建的现代化 AI 智能体管理平台，为 OpenClaw Gateway 提供完整的 Web 管理界面。通过直观的可视化操作，用户可以轻松管理 AI 智能体、会话、模型、频道、技能等核心功能。

### 核心功能

| 功能模块 | 说明 |
|---------|------|
| 仪表盘 | 系统运行状态总览 |
| 在线对话 | 与 AI 智能体实时交互 |
| 会话管理 | 管理和监控所有会话 |
| 记忆管理 | 管理智能体的记忆文档 |
| 任务计划 | 创建和管理定时任务 |
| 模型管理 | 配置和管理 AI 模型 |
| 频道管理 | 配置多渠道支持 |
| 技能管理 | 安装和管理技能插件 |
| 多智能体 | 创建和管理多个智能体 |
| 智能体工坊 | 多智能体协作空间 |
| 虚拟公司 | 可视化办公场景 |
| 远程终端 | 远程命令行操作 |
| 远程桌面 | 远程桌面控制 |
| 文件浏览器 | 工作区文件管理 |
| 系统监控 | 系统资源监控 |
| 系统设置 | 系统配置管理 |

## 快速入门

### 1. 系统登录

#### 访问系统
打开浏览器，访问 `http://localhost:10001`（或您的服务器地址）。

#### 登录步骤
1. 输入用户名（默认：`WKP`）
2. 输入密码（首次登录请修改）
3. 点击"登录"按钮

![登录界面](../images/登录.png)

### 2. 进入仪表盘

登录成功后，自动进入仪表盘页面。

#### 仪表盘功能
- **运行总览**: 显示系统关键指标
- **Token 趋势**: 查看 Token 使用量变化
- **会话统计**: 显示活跃会话数量
- **实时监控**: 实时事件流
- **分布统计**: Top 模型/渠道/工具分布

![仪表盘](../images/仪表盘.png)

### 3. 首次使用建议

1. **检查系统状态**: 确认所有服务正常运行
2. **配置模型**: 添加 AI 模型配置
3. **创建智能体**: 创建第一个 AI 智能体
4. **测试对话**: 使用在线对话功能测试
5. **配置频道**: 如需多渠道支持，配置相应频道

## 功能详解

### 1. 在线对话 (Chat)

#### 功能说明
提供与 AI 智能体的实时聊天交互界面。

#### 操作步骤
1. 点击左侧菜单"在线对话"
2. 选择智能体（默认使用默认智能体）
3. 输入消息，按 Enter 发送
4. 查看 AI 回复

#### 斜杠命令
| 命令 | 说明 |
|------|------|
| `/new` | 创建新会话 |
| `/skill` | 查看可用技能 |
| `/model` | 切换模型 |
| `/status` | 查看会话状态 |
| `/subagents` | 查看子智能体 |

#### 常用语快捷回复
- 点击输入框上方的常用语按钮
- 选择预设消息快速发送

![在线对话](../images/在线对话.png)

---

### 2. 会话管理 (Sessions)

#### 功能说明
管理和监控所有会话，包括创建、重置、删除等操作。

#### 会话列表
- 查看所有会话
- 按状态筛选（活跃/已结束）
- 按智能体筛选
- 按时间排序

#### 会话操作
1. **查看详情**: 点击会话进入详情
2. **重置会话**: 清除会话历史
3. **删除会话**: 永久删除会话
4. **导出记录**: 导出消息记录

#### 筛选功能
- 按智能体名称筛选
- 按创建时间筛选
- 按状态筛选

![会话管理](../images/会话管理.png)

---

### 3. 记忆管理 (Memory)

#### 功能说明
管理智能体的记忆文档，包括 AGENTS、SOUL、IDENTITY、USER 等核心文档。

#### 文档类型
| 文档名 | 说明 |
|--------|------|
| AGENTS.md | 角色定位和职责 |
| SOUL.md | 人格内核和思维模式 |
| IDENTITY.md | 身份信息 |
| USER.md | 用户画像 |
| TOOLS.md | 工具使用指南 |

#### 编辑操作
1. 选择智能体
2. 选择要编辑的文档
3. 在 Markdown 编辑器中修改内容
4. 点击"保存"按钮

#### 模板片段
- 点击工具栏的模板按钮
- 插入常用片段

![记忆管理](../images/记忆管理.png)

---

### 4. 任务计划 (Cron)

#### 功能说明
创建和管理定时任务，支持 Cron 表达式、固定间隔、指定时间等多种触发方式。

#### 创建任务
1. 点击"新建任务"按钮
2. 填写任务名称和描述
3. 选择触发方式：
   - Cron 表达式
   - 固定间隔
   - 指定时间
4. 配置任务内容
5. 保存任务

#### Cron 可视化编辑器
- 选择频率（每天/每周/每月等）
- 设置具体时间
- 实时预览表达式
- 一键复制

#### 任务模板
- 晨报模板
- 健康巡检模板
- 周报模板
- 自定义模板

#### 批量操作
- 批量启用/禁用
- 批量删除

![计划任务](../images/计划任务.png)

---

### 5. 模型管理 (Models)

#### 功能说明
配置和管理 AI 模型，支持多模型渠道配置。

#### 添加模型
1. 点击"添加模型"按钮
2. 选择模型渠道（如 Qwen、Claude 等）
3. 填写 API Key（安全存储）
4. 配置模型参数
5. 点击"探测"测试连接
6. 保存配置

#### 模型设置
- 设置默认模型
- 配置套餐（如 Coding 套餐）
- 启用/禁用模型

#### API Key 安全
- 脱敏显示
- 仅在更新时提交
- 加密存储

![模型管理](../images/模型管理.png)

### 6. 频道管理 (Channels)

#### 功能说明
配置和管理多渠道支持，包括 QQ、飞书、钉钉、企业微信等。

#### 支持的渠道
- QQ
- 飞书
- 钉钉
- 企业微信

#### 配置步骤
以飞书为例：
1. 选择"飞书"频道
2. 点击"安装"按钮
3. 按照指引完成授权
4. 配置频道参数
5. 保存配置

#### 渠道状态
- 在线：正常服务
- 离线：需要检查配置
- 错误：查看错误详情

![频道管理](../images/频道管理.png)

#### Chat 可见性
- 控制技能是否在聊天中可见

## 常见问题

### Q1: 无法登录系统

**可能原因**:
1. 用户名或密码错误
2. Gateway 连接异常
3. 浏览器缓存问题

**解决方案**:
1. 检查用户名密码是否正确
2. 检查 Gateway 服务是否运行
3. 清除浏览器缓存后重试

### Q2: 会话创建失败

**可能原因**:
1. 模型配置错误
2. 网络连接问题
3. Gateway 服务异常

**解决方案**:
1. 检查模型配置是否正确
2. 检查网络连接
3. 查看系统日志

### Q3: 定时任务不执行

**可能原因**:
1. Cron 表达式错误
2. 任务被禁用
3. 系统时间不正确

**解决方案**:
1. 使用可视化编辑器验证表达式
2. 检查任务状态是否为启用
3. 检查服务器时间

### Q4: 模型探测失败

**可能原因**:
1. API Key 无效
2. 网络连接问题
3. 模型渠道配置错误

**解决方案**:
1. 检查 API Key 是否正确
2. 检查网络连接
3. 确认模型渠道配置

### Q5: 频道配置失败

**可能原因**:
1. 授权过期
2. 配置参数错误
3. 权限不足

**解决方案**:
1. 重新授权
2. 检查配置参数
3. 确认应用权限
