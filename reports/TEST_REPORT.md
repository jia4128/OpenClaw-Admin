## 5. 安全测试

### 5.1 npm audit 安全扫描结果

#### 后端安全漏洞 (openclaw-admin)

| 严重级别 | 数量 | 受影响包 | 漏洞描述 |
|---------|------|---------|---------|
| 🔴 **High** | 6 | tar, sqlite3, node-gyp | 任意文件创建/覆盖，路径遍历 |
| 🟡 **Low** | 2 | @tootastnate/once | 控制流作用域错误 |

**详细漏洞**:
1. **tar <= 7.5.10** (High)
   - GHSA-34x7-hfp2-rc4v: 任意文件创建/覆盖
   - GHSA-8qq5-rm4j-mr97: 符号链接 poisoning
   - GHSA-83g3-92jg-28cx: 硬链接路径遍历
   - 影响：sqlite3 -> node-gyp -> make-fetch-happen -> tar

2. **@tootastnate/once < 3.0.1** (Low)
   - GHSA-vpq2-c234-7xj6: 控制流作用域错误
   - 影响：http-proxy-agent -> make-fetch-happen

**修复建议**:
```bash
npm audit fix --force
# 注意：这将升级 sqlite3 到 6.0.1，是破坏性变更
```

#### 前端安全漏洞 (openclaw-admin-frontend)

| 严重级别 | 数量 | 受影响包 | 漏洞描述 |
|---------|------|---------|---------|

**详细漏洞**:
